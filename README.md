# Planilha de Controle de Gastos — Landing Page

Landing page (página de vendas) da **Planilha de Controle de Gastos**, um produto digital para organização financeira pessoal compatível com Excel e Google Sheets.

## Conteúdo

- `index.html` — página completa, autossuficiente (HTML + CSS inline, sem dependências de build).

## Como visualizar

Abra o `index.html` direto no navegador, ou sirva a pasta com qualquer servidor estático:

```bash
npx serve .
```

## Deploy

Por ser um único arquivo estático, pode ser publicado em GitHub Pages, Vercel, Netlify ou qualquer hospedagem de sites estáticos.
